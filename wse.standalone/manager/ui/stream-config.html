Event Listener Examples
Examples:
  • OverlayEvent draws boxes/labels on video
  • Id3Event writes timed metadata into the stream
  • LogFileEvent writes detection data to logs
  • WebhookEvent2 sends detection data to an external HTTP endpoint
  • ObjectTracking applies tracking and region logic, then triggers outputs based on movement, count, time spent in one location, and similar rules
How Matching Works
Use this stream matching configuration rule to choose which incoming streams VIF should analyze.
  • Application must match exactly.
  • Stream Name can be an exact name or a regex pattern.
  • Example: Application live and Stream Name traffic.* will match the ingest stream rtmp://my-wowza-streaming-engine.com/live/traffic_cam1.
