Right version/architecture/folder is decided in TJLoader-unix.java

Files are cherry-picked from apt install of libturbojpeg
install with:
	apt install libturbojpeg

Files come from:
	/usr/lib/x86_64-linux-gnu/libturbojpeg.so.0.2.0
or:
	/usr/lib/aarch64-linux-gnu/libturbojpeg.so.0.2.0

No windows support at this time, will need to find the right dll (c:\libjpeg-turbo[-gcc][64]\bin\libturbojpeg.dll)
